# Oxxito

Sistema distribuido de ventas para una cadena de tiendas de conveniencia tipo Oxxo.

## Tecnologías definidas

- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express + TypeScript
- Base de datos distribuida: Elasticsearch
- Visualización técnica: Kibana
- Infraestructura: Docker Compose

## Estructura del proyecto

- frontend/: interfaz web del sistema
- backend/: API principal del sistema
- elastic/: configuración de Elasticsearch y Kibana
- scripts/: scripts para crear índices y cargar datos
- docs/: documentación técnica del proyecto

## Objetivo

Desarrollar una aplicación distribuida que permita registrar ventas, manejar inventario, restringir acceso por sucursal y generar reportes de ventas usando Elasticsearch como base de datos distribuida.
